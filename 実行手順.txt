Face Practice 実行手順
======================

1. ZIPファイルを展開する

wink_app_distribution.zipを右クリックし、
「すべて展開」を選択します。


2. PowerShellで展開したフォルダへ移動する

cd C:\展開した場所\wink_app_distribution


3. 仮想環境を作成する

python -m venv venv


4. 仮想環境を有効化する

.\venv\Scripts\Activate.ps1

実行できない場合は、次を実行します。

Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

その後、もう一度仮想環境を有効化してください。


5. 必要なライブラリをインストールする

pip install -r requirements.txt


6. 環境設定ファイルを作成する

Copy-Item .env.local .env


7. Flaskアプリを起動する

flask run


8. ブラウザでアプリを開く

http://127.0.0.1:5000


9. アプリを終了する

PowerShellで Ctrl + C を押します。


注意事項
--------

・初回起動時にwink_app.dbが自動作成されます。
・ユーザー登録時にログインIDが自動発行されます。
・発行されたIDは、パスワード認証または顔認証で使用します。
・顔認証AIの初回読み込みにはインターネット接続が必要です。
・カメラの使用を確認された場合は「許可」を選択してください。
